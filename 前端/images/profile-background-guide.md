## ✅ 已完成的 CSS 配置

### 背景设置（`profile.css`）

```css
.main-content {
    background-image: url('../images/profile-background.jpg');
    background-size: cover;           /* 全屏覆盖 */
    background-position: center;      /* 居中对齐 */
    background-repeat: no-repeat;     /* 不重复 */
    background-attachment: fixed;     /* 固定背景（视差效果）*/
}
```

### 叠加层设置

```css
background: linear-gradient(135deg, 
    rgba(255, 255, 255, 0.94) 0%,      /* 左上角：94% 白色 */
    rgba(240, 253, 244, 0.92) 50%,     /* 中间：92% 浅绿色 */
    rgba(238, 242, 255, 0.94) 100%     /* 右下角：94% 浅紫色 */
);
```

### 层级结构

- **z-index: 0** - 背景叠加层
- **z-index: 1** - 表单卡片、标题、消息提示
- **z-index: 2** - 输入框、按钮等交互元素

## 🔧 自定义调整

### 如果图案太明显（影响表单阅读）

编辑 `css/profile.css` 第 14-20 行：

```css
/* 增加叠加层不透明度 */
background: linear-gradient(135deg, 
    rgba(255, 255, 255, 0.96) 0%,      /* 改为 0.96 */
    rgba(240, 253, 244, 0.94) 50%,     /* 改为 0.94 */
    rgba(238, 242, 255, 0.96) 100%     /* 改为 0.96 */
);
```

### 如果图案不够明显

```css
/* 减少叠加层不透明度 */
background: linear-gradient(135deg, 
    rgba(255, 255, 255, 0.90) 0%,      /* 改为 0.90 */
    rgba(240, 253, 244, 0.88) 50%,     /* 改为 0.88 */
    rgba(238, 242, 255, 0.90) 100%     /* 改为 0.90 */
);
```

### 取消视差滚动效果

删除或注释掉第 9 行：

```css
/* background-attachment: fixed; */
```

### 让图案平铺（而不是拉伸）

修改第 6-7 行：

```css
background-size: auto;        /* 改为 auto */
background-repeat: repeat;    /* 改为 repeat */
```

## ❓ 常见问题

**Q: 图案不显示？**
- 检查文件名：`profile-background.jpg`（小写）
- 检查路径：`前端/images/` 文件夹
- 强制刷新：`Ctrl + F5`

**Q: 输入框点不了？**
- 已在 CSS 中设置 `z-index: 1` 确保可交互
- 检查浏览器控制台是否有错误（F12）

**Q: 图案太花哨，影响阅读？**
- 调整叠加层透明度（参考上方"自定义调整"）
- 建议值：0.94 - 0.96 之间

**Q: 在移动端显示不正常？**
- 图案会自动缩放适配
- 如需调整，可修改 `background-size` 属性
