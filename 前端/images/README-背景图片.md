## 🔧 自定义调整

### 如果叠加层太透明（文字不清楚）：

编辑 `css/modern-design.css` 第 197 行附近：

```css
/* 增加不透明度（数值越大越不透明） */
background: linear-gradient(135deg, 
    rgba(255, 255, 255, 0.95) 0%,      /* 改为 0.95 */
    rgba(240, 253, 244, 0.93) 50%,     /* 改为 0.93 */
    rgba(238, 242, 255, 0.95) 100%     /* 改为 0.95 */
);
```

### 如果叠加层太不透明（图片不明显）：

```css
/* 减少不透明度 */
background: linear-gradient(135deg, 
    rgba(255, 255, 255, 0.88) 0%,      /* 改为 0.88 */
    rgba(240, 253, 244, 0.86) 50%,     /* 改为 0.86 */
    rgba(238, 242, 255, 0.88) 100%     /* 改为 0.88 */
);
```

### 如果想取消固定背景效果：

删除或注释掉第 193 行：
```css
/* background-attachment: fixed; */
```

## ❓ 常见问题

**Q: 图片不显示？**
- 检查文件名是否为 `hero-background.jpg`（小写）
- 检查文件是否在 `images/` 文件夹中
- 按 `Ctrl + F5` 强制刷新浏览器

**Q: 图片模糊？**
- 使用高分辨率图片（建议至少 1920x1080）
- 图片格式建议 JPG 或 PNG

**Q: 图片太亮或太暗？**
- 调整 CSS 中的 `rgba` 值（参考上方"自定义调整"）
