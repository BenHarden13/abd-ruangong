# 智能饮食与健康管理平台 - 启动说明

## 🚀 快速启动

### 1. 启动后端服务器（Spring Boot）
```bash
cd 后端
java -jar target/hub-1.0.0.jar
```
**端口**: `8080`  
**API地址**: `http://localhost:8080/api`

### 2. 启动前端服务器（Node.js）
```bash
cd 前端
node server.js
```
**端口**: `3000`  
**访问地址**: `http://localhost:3000`

---

## 📝 重要说明

### 端口配置
- **后端API**: `localhost:8080`
- **前端服务**: `localhost:3000`

### 数据同步机制
系统支持**双重数据存储**：

1. **后端数据库优先**（H2内存数据库）
   - 当后端运行时，数据自动同步到数据库
   - 支持跨浏览器数据访问

2. **本地存储备份**（LocalStorage）
   - 后端不可用时，自动使用本地存储
   - 确保数据不会丢失

### 使用流程
1. **创建健康档案**
   - 访问 `个人健康档案` 页面
   - 填写身高、体重、年龄等基本信息
   - 点击"保存档案"

2. **查看健康仪表盘**
   - 自动跳转或点击 `健康仪表盘`
   - 查看BMI、BMR、TDEE等健康指标
   - 查看营养素分配和建议

3. **获取饮食推荐**
   - 点击 `饮食推荐` 页面
   - 系统根据您的健康目标推荐食谱

---

## 🔧 故障排除

### 问题：仪表盘无法同步数据
**解决方案**：
1. 确认**后端服务**正在运行（8080端口）
2. 确认**前端服务**正在运行（3000端口）
3. 检查浏览器控制台是否有错误信息
4. 尝试清除浏览器缓存和LocalStorage

### 问题：端口被占用
**解决方案**：
- 关闭占用8080或3000端口的其他程序
- 或修改配置文件中的端口号

### 问题：后端无法启动
**解决方案**：
```bash
# 重新编译
cd 后端
mvnw clean package
java -jar target/hub-1.0.0.jar
```
